# ¡Vamos! · Spaans oefenen

Persoonlijke Spaans-leerapp: woordjes met spaced repetition, vertaalzinnetjes, grammatica-toetsjes, spiekbrief, dagdoel en streak. Meerdere profielen met elk een eigen leerpad (niveautest ingebouwd).

De app is één zelfstandig HTML-bestand en draait via GitHub Pages. Voortgang wordt lokaal in de browser bewaard.
